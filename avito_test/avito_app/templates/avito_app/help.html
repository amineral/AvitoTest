API info

/api/client : Clients List

/api/client/id : Client Details

/api/bankoper : bank operation list (depo/draw)

/api/bankoper/id : bank operation details (depo/draw)

/api/depo : put money on clients balance

Required params: id=id, value=value

/api/draw : draw money from clients balance

Required params: id=id, value=value

/api/operation : Operations List

/api/operation/id : Operation Details

/api/transaq : Transaction operation

Required params: from=id, to=id, value=integer. Optional: currency="RUB"(default)

Returns: ERROR: error details / SUCCESS: operation details as JSON
