API info

http://api/client : Clients List

http://api/client/id : Client Details

http://api/operation : Operations List

http://api/operation/id : Operation Details

http://api/transaq : Transaction operation

Required params: from=id, to=id, value=integer. Optional: currency="RUB"(default)

Returns: ERROR: error details / SUCCESS: operation details as JSON
